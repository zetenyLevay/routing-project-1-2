## Prerequisites

Before you begin, ensure you have met the following requirements:

- Java JDK 23 or higher installed
- Apache Maven 3.9.10 or higher installed
- Git (optional, for cloning the repository)

### Checking Prerequisites

```bash
# Check Java version
java -version

# Check Maven version
mvn -version
```
